# RNS

Java bytecode assembler and disassembler.

## Errors and warnings

RNS will report errors and warnings when it encounters issues in the assembly code. RNS is designed to allow
produce
any bytecode, even if it is invalid. This means that RNS will not prevent you from writing code that may not work.

Hovewer, I don't want to leave you in the dark about potential issues in your code, even if they are intentional.
RNS will report errors and warnings to help you identify and fix potential problems in your code.
